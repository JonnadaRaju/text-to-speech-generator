# Telugu Text-to-Speech Generator

A system that converts Telugu text files into audio using the Sarvam AI Text-to-Speech API.

## Features
- Upload `.txt` files containing Telugu text.
- Generate high-quality audio using Sarvam AI's TTS engine.
- Download and list generated audio files.
- Command-line interface for easy interaction.

## Tech Stack
- **Backend:** FastAPI, Python, Requests, Pydantic, aiofiles
- **CLI:** Typer, Requests
- **API:** Sarvam AI TTS API

## Setup Instructions

1.  **Clone the repository:**
    ```bash
    git clone <repository_url>
    cd text-to-speech-generator
    ```

2.  **Create a virtual environment and install dependencies:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```

3.  **Configure API Key:**
    Create a `.env` file in the root directory and add your Sarvam AI API key:
    ```env
    SARVAM_API_KEY=your_sarvam_api_key_here
    ```

4.  **Start the Backend Server:**
    ```bash
    python main.py
    ```
    The server will start at `http://127.0.0.1:8000`.

## CLI Usage

Run the CLI using the `cli.py` script:

### Upload Text
```bash
python cli.py upload-text sample.txt
```

### List Uploaded Texts
```bash
python cli.py list-texts
```

### Generate Audio
```bash
python cli.py generate-audio sample.txt
```

### List Generated Audio
```bash
python cli.py list-audio
```

### Download Audio
```bash
python cli.py download-audio sample_d2d56963.mp3 --output-path output.mp3
```

## API Documentation

The backend provides the following endpoints:

- `POST /texts/upload`: Upload a text file.
- `GET /texts`: List uploaded text files.
- `POST /audio/generate`: Generate audio from a text file.
- `GET /audio`: List generated audio files.
- `GET /audio/{filename}`: Download an audio file.

Interactive API documentation is available at `http://127.0.0.1:8000/docs`.
